# Chatbot-
# 🤖 Voice Chat Bot (Python)

## 📌 Description
This project is a simple **Voice-Based Chat Bot** built using Python.  
It can listen to user voice commands, respond using speech, and perform basic tasks like opening websites.

---

## 🚀 Features
- 🎤 Voice recognition using microphone
- 🗣️ Text-to-speech response
- 🌐 Opens websites like YouTube and Instagram
- 💬 Basic conversation (greetings, name, etc.)
- 🔁 Continuous listening loop

---

## 🛠️ Technologies Used
- Python 🐍
- SpeechRecognition 🎤
- pyttsx3 🔊 (Text-to-Speech)
- Webbrowser 🌐

---

## ▶️ How to Run

1. Install required libraries:
```bash
pip install SpeechRecognition pyttsx3 pyaudio
```

2. Clone the repository:
```bash
git clone https://github.com/your-username/voice-chatbot.git
```

3. Navigate to the project folder:
```bash
cd voice-chatbot
```

4. Run the program:
```bash
python chatbot.py
```

---

## 🎮 How It Works
- The system listens to your voice command
- Converts speech to text
- Matches command with predefined responses
- Speaks back the response
- Can open websites based on command

---

## 🧠 Supported Commands
- "Hello"
- "How are you"
- "Name"
- "YouTube"
- "Instagram"
- "Bye"
